
# Sitio estático — RIOAZA MATAS S.A. de C.V.

Estructura lista para subir a GitHub y publicar en GitHub Pages o Netlify.

## Publicar en GitHub Pages
1. Crea un repositorio y sube esta carpeta.
2. Ve a **Settings → Pages** y elige la rama `main` y carpeta `/root`.
3. Tu sitio quedará disponible en `https://<usuario>.github.io/rioaza-matas-site/`.

## Publicar en Netlify
1. Crea cuenta en Netlify y arrastra esta carpeta al panel.
2. Configura dominio y listo.

## Personalizar
- Reemplaza `assets/img/logo.png` y `assets/img/hero.jpg` por tus imágenes.
- Cambia la etiqueta `<link rel="canonical">` y metadatos OG a tu dominio.
- Edita el `mailto:` del formulario en `index.html`.
